Философия разработки.

Основополагающие принципы, гарантирующие качество, надежность и эффективность разработки.

Принципы SOLID, DRY, KISS, YAGNI.

Строгое следование принципам SOLID, DRY, KISS и YAGNI для создания чистого, поддерживаемого, масштабируемого и легко тестируемого кода Android-приложений.

Методология Agile (Scrum, Kanban).

Применение гибких методологий разработки (Scrum, Kanban) для обеспечения итеративного процесса, быстрой реакции на изменения и тесного взаимодействия в команде.

Безопасность приложений и данных.

Использование ProGuard/R8 для обфускации и минимизации кода, защита данных с помощью SharedPreferences, EncryptedSharedPreferences, Keystore, соблюдение требований GDPR.

Архитектура приложения.

Выбор и реализация архитектурного подхода для обеспечения масштабируемости, тестируемости и поддерживаемости Android-приложений.

Выбор архитектурного паттерна.

Обоснованный выбор между MVVM, VIPER, Clean Swift или Coordinator, исходя из сложности проекта и требований к масштабируемости.

Реализация MVVM (Model-View-ViewModel).

Использование RxSwift или Combine для реактивного программирования, data binding для синхронизации данных между View и ViewModel.

Реализация VIPER (View, Interactor, Presenter, Entity, Router).

Строгое разделение ответственности между компонентами, модульная структура для повышения масштабируемости и упрощения тестирования.

Реализация Clean Swift (Clean Architecture).

Максимальная независимость от фреймворков и UI, использование Use Cases для описания бизнес-логики приложения.

Технологический стек.

Владение и применение современных языков программирования, фреймворков и инструментов для iOS-разработки.

Языки программирования: Swift и Objective-C.

Использование Swift как основного языка разработки, Objective-C – для поддержки legacy-кода и низкоуровневых операций.

UI-фреймворки: UIKit и SwiftUI.

Разработка UI с помощью UIKit (императивный подход) или SwiftUI (декларативный), с использованием Auto Layout и Size Classes.

Сетевое взаимодействие: URLSession, Alamofire, Moya.

Реализация асинхронных сетевых запросов, обработка ответов и ошибок, поддержка REST API и GraphQL, использование NSURLProtocol.

Работа с данными: Core Data, Realm, SQLite.

Локальное хранение данных с использованием Core Data, Realm или SQLite, а также UserDefaults и Property Lists для простых данных.

Тестирование и отладка.

Обеспечение качества кода и приложения с помощью различных методов тестирования и инструментов отладки.

Модульное тестирование (Unit Testing).

Написание unit-тестов с использованием XCTest, Quick, Nimble, применение TDD, mocking и stubbing для изоляции компонентов.

UI-тестирование (UI Testing).

Автоматизация тестирования пользовательского интерфейса с помощью XCTest UI Testing или EarlGrey, использование Accessibility Identifiers.

Отладка приложения.

Использование Xcode Debugger и LLDB для пошаговой отладки, установки точек останова и анализа состояния приложения.

Профилирование приложения.

Использование Xcode Instruments (Time Profiler, Allocations, Leaks) для выявления проблем производительности и утечек памяти.

Дополнительные навыки.

Владение инструментами и методологиями, повышающими эффективность и качество разработки.

Система контроля версий Git.

Использование Git для контроля версий, работа с репозиториями GitHub, GitLab, Bitbucket, ветвление, слияние, разрешение конфликтов.

Паттерны проектирования.

Применение паттернов Singleton, Factory, Adapter, Observer, Delegate, Coordinator, MVC для улучшения структуры и поддерживаемости кода.

Управление зависимостями.

Интеграция сторонних библиотек и фреймворков с помощью CocoaPods, Carthage или Swift Package Manager.

Непрерывная интеграция и доставка (CI/CD).

Автоматизация сборки, тестирования и развертывания приложения с использованием Jenkins, Travis CI, CircleCI, GitLab CI или Fastlane.

Опыт разработки.

Подтвержденный опыт разработки iOS-приложений различной сложности и направленности.

Опыт работы.

Многолетний опыт коммерческой разработки, участие в проектах различного масштаба в продуктовых и аутсорсинговых компаниях.

Разработка приложений разной сложности.

Опыт создания как простых приложений (утилиты), так и сложных систем (социальные сети, e-commerce) с большим количеством экранов.

Интеграция с API и сервисами.

Опыт интеграции с REST API, GraphQL, Firebase, а также со сторонними SDK (платежные системы, аналитика).

Оптимизация производительности.

Профилирование приложений, выявление и устранение узких мест, оптимизация работы с памятью, сетью и UI для повышения отзывчивости.

Коммуникация.

Эффективное взаимодействие в команде и с заказчиком для достижения общих целей.

Командная работа.

Эффективное взаимодействие с дизайнерами, тестировщиками, аналитиками и менеджерами в кросс-функциональной команде, участие в Agile-митингах.

Коммуникативные навыки.

Ясное и четкое изложение мыслей, аргументация своей позиции, составление технической документации, проведение презентаций.

Code Review.

Проведение code review, предоставление конструктивной обратной связи, улучшение качества кода и обмен знаниями в команде.

Долгосрочное сотрудничество.

Поддержка и развитие проекта, масштабирование, внесение изменений, выпуск обновлений.
